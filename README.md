# UrbanKart E-commerce

This is a full-stack e-commerce web application built using the MERN stack (React, Node.js, Express, MongoDB).

## Features
- Browse products
- Add to cart
- User authentication
- Responsive UI

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB

## How to Run

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm start  

---

This project was built to understand how real-world full-stack applications work and how frontend and backend interact.
